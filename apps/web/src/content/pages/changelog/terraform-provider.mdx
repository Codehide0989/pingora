We are excited to announce the release of the Terraform provider for Pingora.
This provider allows you to manage your Grafana Cloud resources using Terraform,
enabling you to define and manage your Pingora resources as code.

Here's the link to the:
[OpenStats Terraform Provider](https://registry.terraform.io/providers/pingora/pingora/latest/docs)
