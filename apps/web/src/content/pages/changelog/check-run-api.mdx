We have added a new API to launch a check with Pingora. You can now test your endpoint's latency programmatically.

It's useful if you want to integrate Pingora with your CI/CD pipeline.

You can read the [API documentation](https://api.pingora.dev/v1#tag/monitor/POST/monitor/:id/trigger).
